## Requirements
- Store birthdays in the existing SQLite-backed memory database.
- Provide MCP tools to add/update, remove, list, and show upcoming birthdays.
- Accept month/day birthdays and optional birth years for age calculations.
- Run a daily local-midnight reminder check through the Discord worker lifecycle.
- Send birthday reminders to the owner without duplicating reminders on worker restart.
- Keep the lookahead behavior in `birthday_upcoming`; automated daily reminders should notify only birthdays happening that local date.
