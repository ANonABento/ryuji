# Local embedding-based memory search

Replace string-match memory search with semantic search via local embeddings. Use ollama embeddings model (mxbai-embed-large) or @xenova/transformers MiniLM. Cache embeddings in SQLite. Re-embed on memory insert. Search returns top-N by cosine similarity. Acceptance: search works without internet, top results semantically relevant, falls back to string search if no embedding service, npx tsc --noEmit passes.
