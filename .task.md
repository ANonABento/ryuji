# Reaction Roles plugin — role assignment via emoji

Implement a new feature that assigns/removes roles when users react/unreact with a configured emoji on a configured message.

Acceptance
- Slash command `/reactionrole add` is owner-only and stores mappings in SQLite.
- Reacting on a tracked message with the configured emoji assigns the configured role.
- Removing that reaction removes the role.
- `npx tsc --noEmit` passes.
