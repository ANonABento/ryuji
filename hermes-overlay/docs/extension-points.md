# Hermes Extension Points For Choomfie

Use these extension points before considering Hermes core patches.

| Need | Preferred Extension Point |
| --- | --- |
| Personality and behavior defaults | Profile files and `SOUL.md` |
| Domain behavior | Skills |
| Structured durable behavior | Plugins |
| Tool access policy | Toolsets |
| Model/provider selection | Hermes provider routing |
| Memory policy | Hermes memory config plus curated import scripts |
| Scheduling | Hermes cron |
| Multi-agent work | Hermes subagents and kanban |
| Discord bridge | Hermes gateway or Bun Choomfie compatibility shell |
| Local launch policy | Overlay scripts |

Patch Hermes core only when an extension point is missing and an upstream contribution is not practical.
