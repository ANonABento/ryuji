# Legacy Choomfie

This directory contains the old Bun/Claude Code Choomfie runtime.

The active direction is Hermes-first:

```bash
choomfie
```

Use the legacy runtime only as a compatibility fallback while Discord UX, tutor, memory, reminders, and voice are ported to Hermes:

```bash
choomfie legacy
choomfie-legacy
```
